Resistor

Resistors limit the flow of charge in a circuit and act as voltage dividers. There are several types of resistors, such as ordinary resistors, sliding resistors (such as potentiometers), and photoresistors. The polarity of a resistor does not matter.

Capacitor

Capacitor is an energy storage element, and its core function is to store and release charge. It can be used for filtering in circuits, removing AC components and maintaining the stability of DC signals. It can also be used for coupling signals and blocking DC signals from passing through, such as audio amplifiers. There are polarized capacitor where the circuit must be connected with the correct polarity and non polarized capacitor.

LED

LED (light emitting diode) is a semiconductor photoelectric device that emits light when current passes through it. It has unidirectional conductivity, and polarity determines its conductivity. It only conducts and emits light when forward biased, and does not emit light when reversed.

Button

A button or switch is a component used to control the on and off of a circuit. When the button is released, the input pin will be in an unstable state (floating), which can easily cause the circuit to be affected by noise and produce random high and low level signals, so a pull-up resistor or a pull-down resistor is needed. In this way, when the button is not pressed, the pull-up resistor makes the input pin to a high level, and the pull-down resistor makes the input pin to a low level.

Transistor

Transistor is a semiconductor electronic component used for signal amplification and switch control. It has three pins, namely base (B) - control current, collector (C) - output current and emitter (E) - current outflow or inflow terminal. Transistor can amplify analog signals in audio. It is usually used as a switch in MCU to drive motors.

Open the link in the ppt, go to the KiCad official website and download the version you like to install KiCad.

Clone or download the fab library on gitlab to your local computer, put it in a location you feel happy with, and then locate the files in this folder in KiCad.

For the schematic symbol library, first open preferences then click manage symbol libraries. The click the folder icon, add existing library to table. Navigate to the cloned or downloaded repository, open the folder and select the .lib file from fab library.

For the footprint library, first open preferences then click manage footprint libraries. The click the folder icon, add existing library to table. Navigate to the cloned or downloaded repository, open the folder and select the .pretty file from fab library.

Then click configure path and add the fab library folder to the table.

To view 3d model, open footprint editor, then select XIAO Generic SocketSMD and click 3d viewer.

First, select XIAO Generic SocketSMD and connect the LED to pin 1. The LED is connected to a resistor, which acts as a current limiter to prevent the LED from burning. Then go to the official website of Xiao RP2040 to check Xiao RP2040. Pin 4 and 5 are SDA and SCL for I2C communication, so connect a 4-pin pin socket, and the other two are vcc and ground. Pin 6 and 7 are TX and RX for UART communication, so connect a 4-pin pin socket, and the other two are vcc and ground. Then use a pull-up resistor and a button to control the LED, and the button is connected to pin11. Then use labels to draw each part separately. Finally, four mount holes were added to facilitate the installation of the PCB board. Need to mark pwr_flag on the vccs and ground. *Remember to check the order of the I2C and UART pin sockets.

Net Classes are used to define the routing rules for different electrical signals to ensure safe spacing between routings, ensure the stability of the circuit and match the manufacturing materials and machines. First open PCB Editor and then open Board Setup to select Net Classes, and then set Track Width, Clearance and Via Size as shown in the figure.

The layers of a PCB are used to define different parts of the board. We will use the copper layer (F.Cu) to place the wires and the Edge. Edge.Cuts Layer is the layer used to define the physical boundaries of the PCB. I also want to use Silkscreen Layer (F.Silkscreen) for marking component names and lettering.

I click update from schematic view and the components will be displayed. I use r to rotate and drag with the mouse to place them where I like. Then I select edge.cuts layer to draw a shape. I made a cloud ☁️ shaped (but actually it looks like marimekko, i have no idea) circuit board. Then it is connection and grounding. Draw wires to connect the corresponding interfaces, draw any shape you like, then draw a huge frame to frame the whole board and press b to make the whole board share the ground. Finally, I selected silkscreen layer and wrote Annie’s First PCB.
